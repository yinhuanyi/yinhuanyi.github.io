    # 一：栅格系统与后台布局

    # 1：栅格系统

    layui 的栅格系统采用业界比较常见的 12 等分规则
    1
    • 栅格布局规则
    1: 采用 layui-row 来定义行，如：<div class="layui-row"></div>

2: 采用类似 layui-col-md* 这样的预设类来定义一组列（column），且放在行（row）内。其中：
	变量md代表的是不同屏幕下的标记（可选值为md1-md12）
	变量* 代表的是该列所占用的12等分数（如6/12），可选值为 1 - 12
	如果多个列的“等分数值”总和等于12，则刚好满行排列。如果大于12，多余的列将自动另起一行。

3: 列可以同时出现最多四种不同的组合，分别是：xs（超小屏幕，如手机）、sm（小屏幕，如平板）、md（桌面中等屏幕）、lg（桌面大型屏幕），以呈现更加动态灵活的布局。

4: 可对列追加类似 layui-col-space5、 layui-col-md-offset3 这样的预设类来定义列的间距和偏移。

5: 最后，在列（column）元素中放入你自己的任意元素填充内容，完成布局！

    # 2：布局容器（个人觉得使用class="layui-container"固定宽度好，布局规整）

    • 将栅格放入一个带有 class="layui-container" 的特定的容器中，以便在小屏幕以上的设备中固定宽度，让列可控，也就是说 class="layui-container"表示的宽度为1140px
    <div class="layui-container">
  <div class="layui-row">
    ……
  </div>
</div###
    1
    2
    3
    4
    5
    • 当然，你还可以不固定容器宽度。将栅格或其它元素放入一个带有 class="layui-fluid" 的容器中，那么宽度将不会固定，而是 100% 适应, 也就是说 class="layui-fluid" 表示的宽度为屏幕宽度
    <div class="layui-fluid">
  ……
</div###

    # 3：列间距

layui-col-space3	列之间间隔 3px
layui-col-space5	列之间间隔 5px
layui-col-space8	列之间间隔 8px
layui-col-space10	列之间间隔 10px
layui-col-space12	列之间间隔 12px
layui-col-space15	列之间间隔 15px
layui-col-space18	列之间间隔 18px
layui-col-space20	列之间间隔 20px
layui-col-space22	列之间间隔 22px
layui-col-space28	列之间间隔 28px

    # 4：使用字体图标

    • 通过对一个内联元素（一般推荐用 i标签）设定 class="layui-icon"，来定义一个图标

    • 例如创建一个笑脸

    <i class="layui-icon layui-icon-face-smile" style="font-size: 30px; color: #1E9FFF;"></i###

    • 除了layui-icon-face-smile，还有140个字体图标可选

    # 5：按钮

    • 按钮有两种定义方式
    1: 标准按钮
<button type="button" class="layui-btn">一个标准的按钮</button>

<a href="http://www.baidu.com" class="layui-btn">一个可跳转的按钮</a>

    # 6：表单

    • 在layui中，在一个容器中设定 class="layui-form" 来标识一个表单元素块，通过规范好的HTML结构及CSS类，来组装成各式各样的表单元素，并通过内置的 form模块 来完成各种交互。依赖加载模块：form （请注意：如果不加载form模块，select、checkbox、radio等将无法显示，并且无法使用form相关功能）

    • 创建一个表单

    {% load staticfiles%}
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1">
  <title>表单的标准使用</title>
  <link rel="stylesheet" href="{% static 'layuiadmin/layui/css/layui.css' %}">
</head>
<body>

<!--外部容器-->
<div class="layui-container">
  <!--栅格布局-->
  <div class="layui-row">
    <!--6等分全占用-->
    <div class="layui-col-md6">

      <!--使用form表单-->
      <div class="layui-form">
        <div class="layui-form-item">
        </div>
        <!--1：用户名-->
        <div class="layui-form-item">
          <label class="layui-form-label">用户名</label>
          <div class="layui-input-block">
            <input type="text" name="username" required  lay-verify="required" placeholder="请输入标题" autocomplete="off" class="layui-input">
          </div>
        </div>

        <!--2：密码-->
        <div class="layui-form-item">
          <label class="layui-form-label">密码</label>

        <!--3：小组-->

          </div>
        </div>

        <!--6：性别-->
        <div class="layui-form-item">
          <label class="layui-form-label">性别</label>
          <div class="layui-input-block">
            <input type="radio" name="sex" value="0" title="男">
            <input type="radio" name="sex" value="1" title="女" checked>
          </div>
        </div>

        <!--7：留言文本-->
        <div class="layui-form-item layui-form-text">
          <label class="layui-form-label">留言文本</label>
          <div class="layui-input-block">
            <textarea name="desc" placeholder="请输入内容" class="layui-textarea"></textarea>
          </div>
        </div>

        <!--8：提交与重置-->
        <div class="layui-form-item">
          <div class="layui-input-block">
            <button class="layui-btn" lay-submit lay-filter="formDemo">立即提交</button>
            <button type="reset" class="layui-btn layui-btn-primary">重置</button>
          </div>
        </div>
        {% csrf_token %}
      </div>
    </div>
  </div>
</div>

// 加载layer模块和form模块
layui.use(['layer','form'], function(){
  // 引入jQuery实现ajax
  var form = layui.form;
  var layer = layui.layer;

  // 加载jQuery
  $=layui.jquery;

  //监听提交按钮，
  form.on('submit(formDemo)', function(data){
    // data.field是获取到表单的数据
    var json = data.field;

    // 可以直接提前字段值
    var username = json.username;
    var password = json.password;
    var group = json.group;
    var write = json.write;
    var read = json.read;
    var run = json.run;
    var active = json.active;
    var sex = json.sex;
    var desc = json.desc;
    var csrf_token = json.csrfmiddlewaretoken;

    console.log(username, password, group, active, sex, desc, csrf_token, write, read, run);

    // ajax提交
    $.ajax({
        url: "{% url 'assets:learning' %}",
        headers: {"X-CSRFToken": csrf_token},
        type: 'post',
        data: json,
        dataType: 'json',
        // 如果请求成功
        success: function (json) {
            console.log(json);
            var code = json.code;
            var msg = json.msg;
            // 让iframe跳转到其他页面
            location.href = "{% url 'assets:index_overview' %}"; //后台主页
        },
        // 如果请求错误
        error: function (xmlhttp, errorText) {
            console.log(xmlhttp);
            console.log(errorText);
            if (xmlhttp.status === "405"){
                console.log("无效的请求方法");
            }else if (xmlhttp.status === "404"){
                console.log("资源不存在");
            }else if (xmlhttp.status === "405"){
                console.log("服务器内部错误");
            } else {
                alert("产生异常，内部错误");
            }
        }
    });

    return false;
  });
});
</script>

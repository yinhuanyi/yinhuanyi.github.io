    # （二）小结此时项目的建设思路(非常重要)

    • 1：在main.js中，创建一个Vue实例，在Vue实例中，引用了整个项目的router路由，render方法，将App这个页面组件渲染成一个虚拟Dom树，基于mount方法将这个虚拟Dom树挂载到index.html页面的id为app的节点上。

    • 2：在App.vue这个组件中，有一个渲染入口<router-view />，所有需要被渲染展示的组件都会在这里展示

    • 3：在router组件中(router/index.js)，配置了项目的路由规则，指定了url 与 component的对应关系，当访问url的时候，就会渲染对应的component，然后将这个component展示在router组件的<router-view />标签中

    • 4：在Home.vue组件中，引入了4个子组件，并且将这个四个子组件显示在Home的template模板中

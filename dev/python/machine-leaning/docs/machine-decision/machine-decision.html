    # (二)信息熵

    • 概念
    熵在信息论中代表：随机变量不确定的度量

熵越大，数据的不确定性越高
熵越小，数据的不确定性越低

    	例如：一个数据集的分类有三类，比率是1/3, 那么这个数据集的熵就是上面的计算公式

def entropy(p):
    return -p * np.log(p) - (1-p)*np.log(1-p)

    # (三)对比sklearn中信息熵的划分，自己实现使用信息熵寻找最优划分的维度和阈值

    • 概念
    	概念：信息熵可以表示当前数据的不确定度，对于决策树来说，根节点拥有全部的数据，在根节点的基础上找到一个维度，找到一个阈值来划分数据，希望划分数据之后，系统的整体信息熵是减小的，对于划分出来的两个节点，再使用相同的方式去寻找特定的维度和阈值进一步划分，使得系统整体的信息熵进一步减小，这样的过程递归下去就形成了决策树
    1

    • 使用信息熵进行划分，且对比sklearn中划分的结果
    # d参数表示维度，value表示阈值， 根据d和v分离样本
def split(X, y, d, value):
    # 寻找索引，样本的某个维度中，小于等于阈值的，返回布尔的向量
    index_a = (X[:, d] <= value)
    index_b = (X[:, d] ### value)
    # 返回指定维度，指定阈值分离样本和分类
    return X[index_a], X[index_b], y[index_a], y[index_b]

# Counter计算序列中元素出现的个数
from collections import Counter
from math import log
# 计算系统的熵值
def entropy(y):
    # 统计每一个类别的个数
    counter = Counter(y)
    res = 0.0
    for num in counter.values():
        # 计算分类的比例
        p = num / len(y)
        # 计算出系统的熵值和
        res += -p * log(p)
    return res

# 这是一个总的输出函数
# 在通过数据样本，寻找这个d维度和value阈值, 和系统最小的entropy值
def try_split(X, y):
    # 初始化的时候设置一个最大的熵，如果找到一个小的熵就将这个熵更新
    best_entropy = float('inf')
    # 初始化维度和阈值
    best_d, best_v = -1, -1
    
    # 开始搜索
    # 对维度进行穷举
    for d in range(X.shape[1]):
        # 对阈值进行穷举, 每两个样本，在d这个维度上，中间的那个值
        # 先对所有样本的d维度的值进行排序，获取排序的索引
        sorted_index = np.argsort(X[:,d])
        
        # 这里为什么从1开始，是因为需要找到上一个值与下一个值的中间值
        # 注意这里是len(X), 不是len(X)+1，目的是寻找阈值
        for i in range(1, len(X)):
            # X[sorted_index[i - 1], d]表示：d这个维度的上一个值 
            # X[sorted_index[i], d]表示：d这个维度的下一个值
            # 两个值的平均值就是这个阈值
            # 需要使得这两个值不相等
            if X[sorted_index[i - 1], d] != X[sorted_index[i], d]: 
                v = (X[sorted_index[i - 1], d] + X[sorted_index[i], d]) / 2
                
                # 那么这个时候d和v的值都有了，就可以调用split对数据集进行划分了
                # 每一次划分都会根据维度和阈值划分左右两颗子树
                X_l, X_r, y_l, y_r = split(X, y, d, v)
                
                # 划分完样本之后，计算系统的熵值
                e = entropy(y_l) + entropy(y_r)
                
                if e < best_entropy:
                    best_entropy, best_d, best_v = e, d, v
                    
    return best_entropy, best_d, best_v


# 对比sklearn第一次划分的结果，在x轴的2.45进行划分
best_entropy, best_d, best_v = try_split(X, y)
print('best_entropy = {}'.format(best_entropy))
print('best_d = {}'.format(best_d))
print('best_v = {}'.format(best_v))

# 查看y1_l的熵值为0，表示，第一次划分左边蓝色的点都被划分到一类，没有任何不确定性

                g = gini(y_l) + gini(y_r)

                if g < best_gini:
                    best_gini, best_d, best_v = g, d, v

    return best_gini, best_d, best_v

    # (八)ID3决策树算法(补充)

    信息增益是机器学习中特征选择的关键指标，而学习信息增益前，需要先了解信息熵和条件熵这两个重要概念。

    • 信息熵

    • 条件熵
    条件熵就是在存在一定条件的情况下，计算的信息熵

1：举例理解条件熵说明：
	条件熵的直观理解：单独计算明天下雨的信息熵H(Y)是2，而条件熵H（Y|X）是0.01（即今天阴天这个条件下，明天下雨的概率很大，确定性很大，信息量就很少），这样相减后为1.99，在获得阴天这个信息后，下雨信息不确定性减少了1.99！是很多的！所以信息增益大！所以是否阴天这个特征信息X对明天下雨这个随机变量Y的来说是很重要的！，那么这个1.99就是信息增益，信息增益越大表示条件越重要。

2：举例计算条件熵
	设样品房数据集样本12份，变量Y为房屋价格，根据价格计算该数据集的香农熵（即信息熵），其中价格高的4个占1/3，价格中等的6个占1/2，价格低的2个占1/6，房屋价格信息熵为：
H(Y) = -1/3log(1/3)-1/2log(1/2)-1/6log(1/6) = 1.0114

    在房屋的面积X这个条件下计算价格Y的条件熵，根据面积X，面积大的4个（价格3高1中）设为a，面积中的3个（价格3中）设为b，面积小的5个（价格1高2中2低）设为c，先分别计算a,b,c条件下的信息熵为：
Ha(x) = -3/4*log(3/4)-1/4*log(1/4) = 0.5623
Hb(x) = -1*log(1) = 0
Hc(x) = -1/5*log(1/5)-2/5*log(2/5)-2/5*log(2/5) = 1.0549
	
再计算a,b,c信息熵按照比例求和，即可得到在条件X条件下，Y的条件熵为:
H(X|Y) = 1/3*Ha(x) + 1/4*Hb(x) + 5/12*Hc(x) = 0.626975

在面积X这个条件下，信息增益可以计算为信息熵减条件熵：
G(Y) = H(Y) - H(X|Y) = 1.0114 - 0.626975 = 0.384425

    import math

# todo: 这里数据集创建的时候，特征的名字使用labels标注了
def createDataSet():
    dataSet = [[1, 1, 'yes'], [1, 1, 'yes'], [1, 0, 'no'], [0, 1, 'no'], [0, 1, 'no']]
    labels = ['no sufacing', 'flippers']
    return dataSet, labels

# todo: 计算信息墒
def calcShannonEnt(dataSet):
    # 获取训练样本数
    numEntries = len(dataSet)

    # 为分类创建字典， 类似于{'yes':2, 'no':4}，目的是计算节点的信息熵
    labelCounts = {}
    # [[1, 1, 'yes'], [1, 1, 'yes']]
    for featVec in dataSet:
        # 获取样本类别
        currentLabel = featVec[-1]

        # 如果发现新的类别，直接在lableCounts里面创建一个
        if currentLabel not in labelCounts.keys():
            labelCounts.setdefault(currentLabel, 0)
        labelCounts[currentLabel] += 1

    # 计算香农墒
    shannonEnt = 0.0
    for key in labelCounts:
        prob = float(labelCounts[key]) / numEntries
        shannonEnt += -prob * math.log(prob)
    return shannonEnt

# todo: 这里是划分特征子集，每一个特征子集就是分支节点，根据特征子集、特征、特征划分值返回一个分支节点的新特征子集
def splitDataSet(dataSet, axis, value):
    retDataSet = []
    # [[1, 1, 'yes'], [1, 1, 'yes'], [1, 0, 'no'], [0, 1, 'no'], [0, 1, 'no']]
    for featVec in dataSet:
        if featVec[axis] == value:
            reduceFeatVec = featVec[:axis]
            reduceFeatVec.extend(featVec[axis + 1:])
            retDataSet.append(reduceFeatVec)
    return retDataSet

# todo: 这里是通过计算信息增益选择最优的划分特征
def chooseBestFeatureToSplit(dataSet):
    # 获取特征个数
    numFeature = len(dataSet[0]) - 1
    # 计算当前数据集的信息熵
    baseEntropy = calcShannonEnt(dataSet)
    # 初始化信息增益
    bestInforGain = 0
    # 初始化最好的特征
    bestFeature = -1

    # 有多少个特征就遍历多少次
    for i in range(numFeature):
        # 拿到第i个特征的所有值，例如： [1,0]
        featList = [number[i] for number in dataSet]
        # 使用集合去重
        uniqualVals = set(featList)
        # 初始化条件熵
        newEntrogy = 0

        # 遍历所有的特征的值
        for value in uniqualVals:
            # 基于数据集，特征，特征值获得一个特征子集，这个特征子集中已经去除了特征i
            subDataSet = splitDataSet(dataSet, i, value)
            # todo: 这里是计算条件熵
            # todo: 下面这行代码是基于在特征i下，不同的取值中，划分的样本数占总样本数的比率
            prob = len(subDataSet) / float(len(dataSet))
            # todo: 计算在特征i下的条件熵
            newEntrogy += prob * calcShannonEnt(subDataSet)

        # todo: 计算按照特征i划分的子节点的信息增益
        infoGain = baseEntropy - newEntrogy


        # 原始数据集划分之后，系统的信息增益最大的就是最好的特征
        if infoGain ### bestInforGain:
            bestInforGain = infoGain
            bestFeature = i
    return bestFeature

# todo: 传递一个类别列表，返回票数最多的类别
def majorityCnt(classList):
    classCount = {}
    for vote in classList:
        if vote not in classCount.keys():
            classCount.setdefault(vote, 0)
        classCount[vote] += 1

    sortedClassCount = sorted(classCount.items(), key=lambda i:i[1], reverse=True)
    return sortedClassCount[0][0]

# todo: 这里的dataset可用是训练集或被分割的特征子集
# 返回的树用字典表示：{'特征1': {1: {特征2:{1:no, 2:yes}},2:{特征3:{1:no, 3:yes}},3:{特征4:{1:no, 2:yes, 3:yes}}}}
def createTree(dataSet, labels):
    # todo: 1：考虑特殊情况，训练集中样本都是一个类别
    # 获取所有样本的类别
    classList = [example[-1] for example in dataSet]
    # 如果样本类别中任意一种类别的个数对于样本数，那么直接输出这个类别
    if classList.count(classList[0]) == len(classList):
        return classList[0]


    # todo: 2：如果节点中的样本已经取出了所有的特征，只剩下类别了，那么就可以取出所有样本的类别，调用majorityCnt函数投票了
    if len(dataSet[0]) == 1:
        return majorityCnt(classList)


    # todo 3：开始迭代创建树
    # 获取最优特征
    bestFeat = chooseBestFeatureToSplit(dataSet)
    # 获取最优特征的解释意义
    bestFeatLabel = labels[bestFeat]
    # 将最优的特征作为树的根节点
    myTree = {bestFeatLabel: {}}
    # todo: 从特征解释意义中将这个特征移除， 由于python都是引用传递，这里复制一份标签列表
    tree_labels = labels[:]
    del (tree_labels[bestFeat])
    # 获取特征子集中，最优特征的所有值
    featValues = [example[bestFeat] for example in dataSet]
    # 使用集合将不同的值去重
    uniqueVals = set(featValues)
    # 变量这个特征的所有值
    for value in uniqueVals:
        # 复制一份特征的解释意义
        subLabels = tree_labels[:]
        # 调用splitDataSet函数，对
        myTree[bestFeatLabel][value] = createTree(splitDataSet(dataSet, bestFeat, value), subLabels)
    return myTree

# {'特征1': {0: {特征2:{1:no, 2:yes}},1:{特征3:{1:no,2:no,3:yes}},2:{特征4:{1:no, 2:yes, 3:yes}}}}
# todo: 训练出来的决策树，特征的意义标签，一个测试样本，根据测试样本特征的值与决策树进行值的划分进行对比，最后数叶子节点的类别就是样本的类别
def classify(inputTree, featLabels, testVec):

    # 获取第一个最优特征
    firstStr = list(inputTree.keys())[0]
    # 获取特征下的字典
    sendDict = inputTree[firstStr]
    # 获取第一个最优特征的索引
    featIndex = featLabels.index(firstStr)
    # 初始化类别标签
    classLabel = None
    # 变量第一个特征的所有的特征值
    for key in sendDict.keys():
        # 如果待测样本的第一个最优特征的值与决策树的第一个最优特征的值相等
        if testVec[featIndex] == key:
            # 如果决策树第一个最优特征的值下面还有特征子集需要划分
            if type(sendDict[key]).__name__ == 'dict':
                # 那再次将这个特征子集作为树，再次调用classify进行迭代
                classLabel = classify(sendDict[key], featLabels, testVec)
            else:
                # 如果不是字典，说明这个特征的值对应的是类别，那么直接将这个类别返回即可
                classLabel = sendDict[key]
    return classLabel

    # 初始化训练集合，特征的标签
    dataSet, feature_labels = createDataSet()
    # 感觉训练集和特征标签创建决策树
    myTree = createTree(dataSet, feature_labels)
    # 通过决策树分类器函数，对样本进行分类
    sample = [0,1]
    r = classify(myTree, feature_labels, sample)
    # 打印样本类别
    print(r)

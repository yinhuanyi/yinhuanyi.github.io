    • 做线性回归目标是解决y(i)真值和y ^ (i) 的差距尽量变小, 那使用什么函数模拟两个y之间的距离呢？ 可以考虑使用绝对值函数，但是绝对值函数不是处处可导函数，因此选择使用两个y的差的绝对值来衡量距离， 那么m个样本的距离就可以表示为下面的公式

    • 将找到的直线方程带入距离公式，就将问题转换成为找到参数a和参数b的值，求解最小值的问题

    • 所有参数学习的算法都是这样的一个套路， 包括线性回归算法、多项式回归算法、逻辑回归算法、SVM、神经网络等

	最小二乘法（又称最小平方法）是一种数学优化技术。它通过最小化误差的平方和寻找数据的最佳函数匹配，利用最小二乘法可以简便地求得未知的数据，并使得这些求得的数据与实际数据之间误差的平方和为最小。最小二乘法还可用于曲线拟合。其他一些优化问题也可通过最小化能量或最大化熵用最小二乘法来表达。

    import numpy as np

class MySimpleLinearRegression1:
    """
    实现简单线性回归算法1
    """
    def __init__(self):
        self.a_ = None
        self.b_ = None

    def fit(self, x_train, y_train):
        '''
        :param x_train:
        :param y_train:
        :return:
        '''
        assert x_train.ndim == 1, \
        'x_train 必须是一维向量'

        assert len(x_train) == len(y_train), \
        'x_train与y_train必须相等'

        # 求向量平均数
        x_mean = np.mean(x_train)
        y_mean = np.mean(y_train)

        # 分子
        numerator = 0.0
        # 分母
        denominator = 0

        # 计算出a参数的分子和分母
        for x_i, y_i in zip(x_train, y_train):
            numerator += (x_i - x_mean) * (y_i - y_mean)
            denominator += (x_i - x_mean) ** 2

        self.a_ = numerator / denominator
        self.b_ = y_mean - self.a_ * x_mean

        return self

    def predict(self, x_predict):
        '''
        :param x_predict: 一维向量
        :return: 一维向量
        '''
        assert x_predict.ndim == 1, \
        'x_predict必须是一维向量'

        assert self.a_ is not None and self.b_ is not None, \
        '{}, {}必须在fit()方法后调用predict()方法'.format(self.a_, self.b_)

        return np.array([self._predict(x) for x in x_predict])

    def _predict(self, x_single):
        return self.a_ * x_single + self.b_

    def __repr__(self):
        return "MySimpleLinearRegression1(a_={}, b_={})".format(self.a_, self.b_)

    __str__ = __repr__

    # (四)使用向量化优化a参数的求解(之前计算a、b参数直接for循环求解的，现在提升算法性能，将a参数的推导式改为向量的数量积)

    • 改造MySimpleLinearRegression2类对a参数的求解
    class MySimpleLinearRegression2:
    """
    实现简单线性回归算法2： 将其a参数的分子和分母进行向量化
    """
    def __init__(self):
        self.a_ = None
        self.b_ = None

    def fit(self, x_train, y_train):
        '''
        :param x_train:
        :param y_train:
        :return:
        '''
        assert x_train.ndim == 1, \
        'x_train 必须是一维向量'

        assert len(x_train) == len(y_train), \
        'x_train与y_train必须相等'

        # 求向量平均数
        x_mean = np.mean(x_train)
        y_mean = np.mean(y_train)

        # 分子
        numerator = 0.0
        # 分母
        denominator = 0
        
        # 将分子和分母的运算转变为向量的数量积
        # 这里必须使用.dot方法，进行数学上的向量的数量积运算, 得到的是一个数
        numerator = (x_train - x_mean).dot(y_train - y_mean)
        denominator = (x_train - x_mean).dot(x_train - x_mean)

        self.a_ = numerator / denominator
        self.b_ = y_mean - self.a_ * x_mean

        return self

    def predict(self, x_predict):
        '''
        :param x_predict: 一维向量
        :return: 一维向量
        '''
        assert x_predict.ndim == 1, \
        'x_predict必须是一维向量'

        assert self.a_ is not None and self.b_ is not None, \
        '{}, {}必须在fit()方法后调用predict()方法'.format(self.a_, self.b_)

        return np.array([self._predict(x) for x in x_predict])

    def _predict(self, x_single):
        return self.a_ * x_single + self.b_

    def __repr__(self):
        return "MySimpleLinearRegression1(a_={}, b_={})".format(self.a_, self.b_)

    __str__ = __repr__

    # (五)线性回归算法好坏的标准评价

    • 标准推导

    • 衡量算法的好坏与m的规模大小相关

    • 均方误差 MSE (Mean Squared Error): 除以M样本数量，让误差与M样本的规模无关

    • 均方根误差 RMSE (Root Mean Squared Error): 让MSE开根号，保证计算出来的量纲和原始的量纲一致

    • 平均绝对误差 MAE (Mean Absolute Error)

    • 在metrics.py模块中加入这三个测评函数
    def mean_squared_error(y_test, y_predict):
    assert len(y_test) == len(y_predict), \
    '预测向量长度必须等于测试向量的长度'
    return np.sum((y_predict - y_test) ** 2) / len(y_test)

def root_mean_squared_error(y_test, y_predict):
    assert len(y_test) == len(y_predict), \
        '预测向量长度必须等于测试向量的长度'
    return sqrt(np.sum((y_predict - y_test) ** 2) / len(y_test))

def mean_absolute_error(y_test, y_predict):
    assert len(y_test) == len(y_predict), \
        '预测向量长度必须等于测试向量的长度'
    return np.sum(np.absolute(y_predict - y_test)) / len(y_test)

    # (六)R Squared 评测算法

    • 计算公式

    • 使用R Squared 评测算法的意义：分子是模型测试的错误，分母是基准模型的错误，分子除以分母的意义在于计算学习到的模型比基准模型犯错误的比率, 那么1-犯错误的比率=没有犯错误的比率， 如何R Squared为负数，那么可能预测数据不具备线性相关性

    • 将R Squared 评测算法进行化简：分子就是MSE测评算法，分母就是测试数据的方差

    • 在metrics.py模块中加入R Squared 函数的计算
    def r2_score(y_test, y_predict):
    return 1 - mean_squared_error(y_test, y_predict) / np.var(y_test)

    # (一)什么是SVM

    • 概念

        对于逻辑回归来说，在样本空间找到一个决策边界，通过决策边界将样本进行分类
    1

    	但是有时候也会出现决策边界不唯一的问题，被称为不适定问题，在逻辑回归中，使用概率函数sigmoid函数，进行建模，形式一个损失函数，最小化损失函数，计算出theta向量，从而求得一条决策边界
    1

    	SVM算法目的是找到一个最优的决策边界是的距离两个类别最近的样本的距离达到最远
	决策边界两边平行的直线就是支撑向量，支撑向量直接的距离就是margin，那么SVM就是要使margin的取值最大化
	这种线性可分的问题叫做Hard Margin SVM,对于线性不可分叫做Soft Margin SVM

    • 修改绘图函数，绘制出支撑向量的直线
    def plot_svc_decision_boundary(model, axis):
    
    x0, x1 = np.meshgrid(
        np.linspace(axis[0], axis[1], int((axis[1]-axis[0])*100)).reshape(-1,1),
        np.linspace(axis[2], axis[3], int((axis[3]-axis[2])*100)).reshape(-1,1)
    )
    X_new = np.c_[x0.ravel(), x1.ravel()]
    
    y_predict = model.predict(X_new)
    zz = y_predict.reshape(x0.shape)
    
    from matplotlib.colors import ListedColormap
    custom_cmap = ListedColormap(['#EF9A9A','#FFF59D','#90CAF9'])
    
    plt.contourf(x0, x1, zz, linewidth=5, cmap=custom_cmap)
    
    
    # 绘制支撑向量的直线
    w = model.coef_[0]
    b = model.intercept_[0]
    # 那么第一根直线就是 w0·x0 + w1·x1 + b = 0
    # 改写一下直线方程：x1 = -w0/w1*x0 - b/w1
    plot_x = np.linspace(axis[0], axis[1], 200)
    # 这两条直线只是截距不一样，是平行的
    up_y = -w[0]/w[1]*plot_x - b/w[1] + 1/w[1]
    down_y = -w[0]/w[1]*plot_x - b/w[1] - 1/w[1]
    
    # 将这两条直线过滤到axis轴尺度的内部, up_index、down_index都是布尔类型的列表
    up_index = (up_y >= axis[2]) & (up_y <= axis[3])
    down_index = (down_y >= axis[2]) & (down_y <= axis[3])
    
    # 绘制图形
    plt.plot(plot_x[up_index], up_y[up_index], color='black')
    plt.plot(plot_x[down_index], down_y[down_index], color='black')

    • 简单回顾正态分布
    1：期望和均值
	虽然都是有平均的概念，但一个很根本的区别在于，期望是随机变量的总体的平均，而平均值是从总体中抽取出来的样本的平均。前者是理论上的值、理想值，后者是现实观察到的统计量。举个例子，掷一枚六面均匀的骰子所得的点数 X，这是个随机变量，X 的期望是 3.5（= [ 1 + 2 + 3 + 4 + 5 + 6 ] / 6 ）。而平均值呢？将多次掷这枚骰子所得的点数求平均——比如掷五次取平均值，每次实验测得的平均值可能与期望 3.5 有差异
	均值（mean value）是针对既有的数值（简称母体）全部一个不漏个别都总加起来，做平均值（除以总母体个数），就叫做均值。此法针对小群体做此加总后除以个数得到均值的方法，是很准确无误的，这个得到的均值是准确的，不会有模糊的概念。但是当这个数群（data group）的数量（numbers）很大很多时，我们只好做个抽样（sampling），并“期望”透过抽样所得到的均值，去预测整个群体的“期望值（expectation value）”。在概率论和统计学中，期望值（或数学期望、或均值，亦简称期望，物理学中称为期待值）是指在一个离散性随机变量试验中每次可能结果的概率乘以其结果的总和

2：正态分布
	若随机变量X服从一个数学期望为μ、方差为σ^2的正态分布，记为N(μ，σ^2)。其概率密度函数为正态分布的期望值μ决定了其位置，其标准差σ决定了分布的幅度。当μ = 0,σ = 1时的正态分布是标准正态分布。

    • 使用高斯核函数将一维映射到二维
    # x是样本的一维值，l是地标点
def gaussian(x, l):
    # γ取值
    gamma = 1.0
    return np.exp(-gamma * (x-l)**2)


# 假设取两个地标值
l1, l2 = -1, 1

# 存储经过核函数转换后的新二维的数据样本
X_new = np.empty((len(x), 2))

for i, data in enumerate(x):
    X_new[i, 0] = gaussian(data, l1)
    X_new[i, 1] = gaussian(data, l2)

print(X_new)

    • 高斯核函数小结

    	高斯核函数的本质是升维，将m*n的数据样本，映射称为了m*m的数据样本，也就是说原本样本只有n个维度，但是经过高斯核函数之后就有m个维度了。那么如果m样本数量非常大的化，那么经过高斯核函数处理之后，样本的维度就是一个无穷维的样本，因此SVM算法使用高斯核函数的训练时间比较长。

    # (八)sklearn中的高斯核函数

    • 高斯函数

    	在高斯函数中有两个参数，一个是u代表均值(期望)，一个是σ(标准差)， u决定了高斯函数的中心轴的位置，对于σ用来描述样本数据发布的情况，σ值越大，函数显得越胖，σ值越小，函数显得越瘦
	对于γ来说，正好与σ成反比，那么γ的值越大，高斯函数越瘦，γ的值越小，高斯函数越胖

    from sklearn.preprocessing import StandardScaler
from sklearn.svm import SVC
from sklearn.pipeline import Pipeline


def RBFKernelSVC(gamma=0.1):
    return Pipeline([
        ('std_scaler', StandardScaler()),
        ('svc', SVC(kernel='rbf', gamma=gamma)),
    ])

rbf_kernel_svc = RBFKernelSVC(gamma=1.0)
rbf_kernel_svc.fit(X, y)

# 绘制决策边界
def plot_decision_boundary(model, axis):
    
    x0, x1 = np.meshgrid(
        np.linspace(axis[0], axis[1], int((axis[1]-axis[0])*100)).reshape(-1,1),
        np.linspace(axis[2], axis[3], int((axis[3]-axis[2])*100)).reshape(-1,1)
    )
    X_new = np.c_[x0.ravel(), x1.ravel()]
    
    y_predict = model.predict(X_new)
    zz = y_predict.reshape(x0.shape)
    
    from matplotlib.colors import ListedColormap
    custom_cmap = ListedColormap(['#EF9A9A','#FFF59D','#90CAF9'])
    
    plt.contourf(x0, x1, zz, linewidth=5, cmap=custom_cmap)
    
    
# 绘图
plot_decision_boundary(rbf_kernel_svc, axis=[-1.5,2.5,-1.0,1.5])
plt.scatter(X[y==0, 0], X[y==0, 1], color='r')

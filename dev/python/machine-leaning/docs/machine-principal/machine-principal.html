    # (一)PCA 主成分分析法基本概念

    • PCA的基本原理

    • 如果找到一个轴，让样本数据落到这个轴上，间距最大，就是我们希望的轴

    • 那么如何来定义样本之间的间距呢？
    	那就是方差，方差的意义在于：考察样本数据的波动性，方差越大说明样本数据的间隔越大，也说明波动性越大；方差越小说明样本数据的间隔越小，也说明波动性越小
    1
    • 方差的计算公式

    • 那么问题就转换为了: 找到一个轴，使得样本空间的所有点映射到这个轴后，方差最大

    • 由于方差的计算需要让每一个样本减去均值，那么现在移动坐标轴，试着将样本的均值定义为0，那么方差公式变为如下公式，这种处理方法叫做demean处理， 因为移动了坐标轴，此时样本的数据也发生了变化(在代码里面只需要让矩阵的每一列减去这一列的均值即可，就完成了demean的操作), 这里的$x_i$是已经是源数据样本映射到($w_1$, $w_2$)向量的新样本了

    • 因为X是矩阵，和数值不一样，这里对X矩阵做取模，运算之后在进行平方

    • 要使得$X_p$的模最大，也就是将$X_i$映射到w向量的方向上，其实就是求解$X_i$ * w点乘运算

    • 这里将w这个向量设置为单位向量，那么||w||=1，那么点乘运算就能够化简为

    • 那么最后PCA主成分分析法问题转换为

    • 这里的$X_i$ · w是两个向量的运算，各项相乘再相加

    • 最终问题转换为目标函数最优化的问题，也就是一个上升问题

    # 对PCA主成分分析法与线性回归问题的区别

    1: 在PCA中，横轴和纵轴都是两个特征， 样本是垂直于w向量
2：在线性回归中，横轴是特征，纵轴是输出标记，使预测结果的MSE尽可能小

    • 经过之前公式的推导，通过梯度上升算法求解方程

    # 传递两个参数，一个是w向量，一个X矩阵, X矩阵必须是均值归0化
def f(w, X):
    return np.sum(X.dot(w)**2) / len(X)

# 偏导数
def df_math(w,X):
    return X.T.dot(X.dot(w)) * 2. / len(X) 

# 使用debug模式验证一下偏导数的推导公式是否正确
def df_debug(w, X, epsilon=0.001):
    res = np.empty(len(w))
    for i in range(len(w)):
        w_1 = w.copy()
        w_1[i] += epsilon
        w_2 = w.copy()
        w_2[i] -= epsilon
        res[i] = (f(w_1, X) - f(w_2, X)) / (2 * epsilon)
    return res

# 给定一个向量，这个向量的模不为1，那么通过这个函数，让这个向量的模为1
def direction(w):
    # np.linalg.norm(w) 求w向量的模，一个向量除以模对于单位向量
    return w / np.linalg.norm(w)

# 梯度上升法计算w
def gradiet_ascent(df, X, initial_w, eta, n_iters=1e4, epsilon=1e-8):
    
    # 注意点1：将initial_w转换为单位向量
    w = direction(initial_w)
    cur_iter = 0
    
    while cur_iter < n_iters:
        gradient = df(w,X)
        last_w = w
        w = w + eta*gradient
        # 将w向量单位化
        w = direction(w)
        
        if abs(f(w, X) - f(last_w, X)) < epsilon:
            break
        
        cur_iter += 1
        
    return w

    • 给定一个X矩阵，计算X矩阵的前n个主成分, 返回一个列表
    # 计算X的前n个主成分, 返回一个列表
def first_n_components(n, X, eta=0.01, n_iters=1e4, epsilon=1e-8):
    # 保证原矩阵数据不变，先拷贝一份
    X_pca = X.copy()
    # demean处理
    X_pca = demean(X_pca)
    # 用于保存各个主成分
    res = []
    for i in range(n):
        # 初始化initial_w
        initial_w = np.random.random(X_pca.shape[1])
        # 计算出第一个主成分
        w = first_component(X_pca, initial_w, eta)
        # 添加到列表中
        res.append(w)
        # 计算下一个矩阵，用于求下一个主成分
        X_pca = X_pca - X_pca.dot(w).reshape(-1, 1) * w
    return res

    # (三)高维数据向低维数据映射(降维)

    • X矩阵是数据样本集，$W_k$是前k个主成分矩阵，$W_k$的k行代表前k个主成分，n列表示每一个主成分的对应n个坐标轴，也就是n维向量。主成分分析法本质是从一个坐标系转换到另外一个坐标系，坐标系的维度不变，只是特征维度被压缩了。因此每一个主成分可以被看做为一个轴，$W_k$可以被看做一个新的坐标系，有k个轴组成

    • 那么对于特征压缩来说，如何将一个矩阵从n维特征转换为k维特征呢？

    • 对于一个样本与一个w主成分进行点乘运算，得到的结果就是这个样本在w这个主成分方向上的模的大小。那么如果一个样本与k个主成分点乘运算，得到的结果就是这个样本在这个k个主成分方向上的对应的大小，那么就有k个数，这个k数组成的向量就是这个样本映射到$W_k$这个坐标系上得到的新的k维的向量，可以认为是一个列向量，那么n比k小，那么这样就完成了一个矩阵从n维向k维转变

    • 那么下面就是降维的公式

    • 也可以通过$X_k$的矩阵与$W_k$矩阵，恢复成原来的$X_m$矩阵，但是这个 $X_m$矩阵与之前的$X$矩阵不一样了

    • 在pycharm中封装自己的PCA降维算法
    import numpy as np

class PCA:
    # n_components是用户指定计算出多少个主成分向量
    def __init__(self, n_components):
        assert n_components >= 1, '主成分个数必须合法'
        self.n_components = n_components
        # 保存计算出来的主成分矩阵
        self.components_ = None

    # 计算self.components_这个主成分矩阵
    def fit(self, X, eta=0.01, n_iters=1e4):
        assert self.n_components <= X.shape[1],  '矩阵的维度必须大于主成分的个数'

        # 对X矩阵进行demean操作
        def demean(X):
            return X - np.mean(X, axis=0)

        def f(w, X):
            return np.sum(X.dot(w) ** 2) / len(X)

        # 偏导数
        def df(w, X):
            return X.T.dot(X.dot(w)) * 2. / len(X)

        # 给定一个向量，这个向量的模不为1，那么通过这个函数，让这个向量的模为1
        def direction(w):
            # np.linalg.norm(w) 求w向量的模，一个向量除以模对于单位向量
            return w / np.linalg.norm(w)

        # 梯度上升法计算w
        def first_component(X, initial_w, eta, n_iters=1e4, epsilon=1e-8):

            # 注意点1：将initial_w转换为单位向量
            w = direction(initial_w)
            cur_iter = 0

            while cur_iter < n_iters:
                gradient = df(w, X)
                last_w = w
                w = w + eta * gradient
                # 将w向量单位化
                w = direction(w)

                if abs(f(w, X) - f(last_w, X)) < epsilon:
                    break

                cur_iter += 1

            return w


        # todo: 开始计算出主成分矩阵
        # demean处理
        X_pca = demean(X)
        # 初始化主成分矩阵， k x n
        self.components_ = np.empty(shape=(self.n_components, X.shape[1]))

        for i in range(self.n_components):
            # 初始化initial_w
            initial_w = np.random.random(X_pca.shape[1])
            # 计算出第一个主成分
            w = first_component(X_pca, initial_w, eta)
            # 添加到列表中
            self.components_[i, :] = w
            # 计算下一个矩阵，用于求下一个主成分
            X_pca = X_pca - X_pca.dot(w).reshape(-1, 1) * w

        return self

    # 将X矩阵进行降维处理
    def transform(self, X):
        assert X.shape[1] == self.components_.shape[1]
        return X.dot(self.components_.T)

    # 将矩阵X进行升维, 与transform正好相反
    def inverse_transform(self, X):
        assert X.shape[1] == self.components_.shape[0]
        return X.dot(self.components_)

    def __repr__(self):
        return 'PCA(n_components={})'.format(self.n_components)

    __str__ = __repr__

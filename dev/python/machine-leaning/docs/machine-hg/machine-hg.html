    # (四)模型的泛化能力

    	线性回归和多项式回归目的就是找到一个模型，当有新的样本的时候，输入到这个模型中， 这个模型可以给出更好的解答。因此，评价一个模型的好坏，使用训练数据做判断是没有意义的，评价一个模型的好坏是评价这个模型的泛化能力有多好。 模型的泛化能力就是指：泛化能力通俗来讲就是指学习到的模型对未知数据的预测能力。在实际情况中，我们通常通过测试误差来评价学习方法的泛化能力

        # 每一次的训练数据样本都不断增多
        algo.fit(X_train[:i], y_train[:i])
        # 预测训练数据
        y_train_predict = algo.predict(X_train[:i])
        train_score.append(mean_squared_error(y_train[:i], y_train_predict))
        # 预测测试数据
        y_test_predict = algo.predict(X_test)
        test_score.append(mean_squared_error(y_test, y_test_predict))
        
    # 绘制随着训练数据增加，训练数据和测试数据均方根误差的变化情况
    plt.plot([i for i in range(1, 76)], np.sqrt(train_score), label='train data')
    plt.plot([i for i in range(1, 76)], np.sqrt(test_score), label='test data')
    plt.legend()
    plt.axis([0, len(X_train) + 1, 0, 4])
    plt.show()

    # 一：链表是动态数据结构

    # (一)：为什么链表是动态数据结构

    • 之前的数组、栈、队列都是静态数据结构，因为他们的扩容是基于resize实现的，扩容和缩容都需要遍历整个底层数组中的元素，因此不适合在大数据量的情况下使用

    • 链表是一个动态数据结果，也是最简单的动态数据结构(通过的学习可以更加深入的理解什么是指针)

    • 链表有清晰的递归逻辑

    # (二)：链表的优缺点

    • 优点
    真正的动态数据结构，不需要处理容量的问题
    1
    • 缺点
    无法随机访问元素
    1

    # (三)：数组与链表对比

    • 是否有连续的内存空间
    数组是一块连续的内内存空间，是静态的数据结构，可以通过索引获取内存中储存的数据

链表的内存空间不是连续的内存空间，必须通过next指针找到下一个元素的地址
    1
    2
    3
    • 是否支持快速查询
    数组支持快速查询

链表不支持快速查询

	// 判断index的合法性
	if index < 0 || index > l.size {
		log.Panicf("index 不合法")

	if index == 0 {
		l.addFirst(e)
		return

	prev := l.head

	for i := 0; i < index-1; i++ {
		prev = prev.next
	}

	//// 创建新的节点
	//node := &Node{e: e}
	//// 让新节点的next指向prev的next
	//node.next = prev.next
	//// 让prev的next指向新节点
	//prev.next = node

	//
	prev.next = &Node{e: e,
		next: prev.next,
	}

}

    # (五)：在链表的头部设置虚拟节点：dummyhead

    • 设置虚拟节点的目的是：由于链表头节点没有前面的节点，因此每次在插入节点的时候，需要判断一下是否插入的是头节点，那么为了解决这个问题，给链表设置一个虚拟节点，这个虚拟节点不存储任何数据，只是为了操作链表的时候方便。那么这个虚拟节点称之为dummyHead节点，链表的第一个节点就是dummyHead节点的next指向的节点

    • 有了dummyHead节点之后，所有的节点都有prev节点了，此时代码修改如下，添加了链表的构造函数，且链表头部和尾部添加元素的方法都将调用insert方法

    // Node 节点
type Node struct {
	e    int   // 数据
	next *Node // 指向的下一个节点
}

// NodeToString 打印节点元素
func NodeToString(node *Node) string {
	nodeByte, err := json.Marshal(node)
	if err != nil {
		log.Panicf(err.Error())
	}
	return string(nodeByte)
}

// LinkedList 链表
type LinkedList struct {
	dummyHead *Node // 链表指向的第一个节点
	size int   // 链表的长度
}

// NewLinkedList 构造函数
func NewLinkedList() *LinkedList {
	return &LinkedList{
		dummyHead: new(Node),
		size: 0,
	}
}

// getSize 获取链表的长度
func (l *LinkedList) getSize() int {
	return l.size
}

// isEmpty 判断链表是否为空
func (l *LinkedList) isEmpty() bool {
	return l.size == 0
}

// insert 列表中插入元素 （插入的核心是找到index节点的前一个节点，并且让新节点的next指向prev的next，让prev的next指向新节点）
func (l *LinkedList) insert(index, e int) {

	// 判断index的合法性
	if index < 0 || index > l.size {
		log.Panicf("index 不合法")
	}

	// prev是链表的dummy节点
	prev := l.dummyHead

	// 不需要考虑插入的是头部还是尾部了
	for i := 0; i < index; i++ {
		prev = prev.next
	}

	// 创建新的节点, 让新节点的next指向prev的next, 让prev的next指向新节点
	prev.next = &Node{e: e,
		next: prev.next,
	}

	l.size++

}

// addLast 向元素末尾添加元素
func (l *LinkedList) addLast (e int) {
	l.insert(l.size, e)
}

// addFirst 在链表的头部添加元素
func (l *LinkedList) addFirst(e int) {
	l.insert(0, e)
}

func main() {

	linkedList := NewLinkedList()
	linkedListSize := linkedList.getSize()
	fmt.Println(linkedListSize)

}

    # (六)：给链表添加get、getFirst、getLast、set、contains、linkedListToString方法

// 链表的get操作, 获取节点的值
func (l *LinkedList) get (index int) int {

	if index < 0 || index >= l.size {
		log.Panicf("index 不合法")
	}

	// 获取第一个节点
	prev := l.dummyHead.next

	// 获取index处的节点
	for i:=0; i<index; i++ {
		prev = prev.next
	}

	return prev.e
}

// getFirst 获取第一个节点的值
func (l *LinkedList) getFirst() int {
	return l.get(0)

    # (七)：给链表删除元素

    • 删除节点思想是：获取被删除节点的前一个节点，将前一个节点的next指向被删除节点的后一个节点。最后让被删除节点的next指向nil，就删除了。

    • 代码实现：代码添加了remove、removeFirst、removeLast方法
    // Node 节点
type Node struct {
	e    int   // 数据
	next *Node // 指向的下一个节点
}

// NodeToString 打印节点元素值
func (n *Node) NodeToString() string {
	return strconv.Itoa(n.e)
}

// LinkedList 链表
type LinkedList struct {
	dummyHead *Node // 链表指向的第一个节点
	size int   // 链表的长度
}

// NewLinkedList 构造函数
func NewLinkedList() *LinkedList {
	return &LinkedList{
		dummyHead: new(Node),
		size: 0,
	}
}

// getSize 获取链表的长度
func (l *LinkedList) getSize() int {
	return l.size
}

// isEmpty 判断链表是否为空
func (l *LinkedList) isEmpty() bool {
	return l.size == 0
}

// insert 列表中插入元素 （插入的核心是找到index节点的前一个节点，并且让新节点的next指向prev的next，让prev的next指向新节点）
func (l *LinkedList) insert(index, e int) {

	// 判断index的合法性
	if index < 0 || index > l.size {
		log.Panicf("index 不合法")
	}

	// prev是链表的dummy节点
	prev := l.dummyHead

	// 不需要考虑插入的是头部还是尾部了
	for i := 0; i < index; i++ {
		prev = prev.next
	}

	// 创建新的节点, 让新节点的next指向prev的next, 让prev的next指向新节点
	prev.next = &Node{e: e,
		next: prev.next,
	}

	l.size++

}

// addLast 向元素末尾添加元素
func (l *LinkedList) addLast (e int) {
	l.insert(l.size, e)
}

// addFirst 在链表的头部添加元素

	l.insert(0, e)
}

	prev := l.dummyHead.next

	// 获取index处的节点

	if index < 0 || index >= l.size {

	}

	prev := l.dummyHead
	for i := 0; i < index; i++ {

	delNode := prev.next

    # (一)：链表实现栈

    • 由于链表这种数据结构，只有操作头部元素的时间复杂度是O(1)，因此非常适合作为栈的底层数据结构，因为栈是先进后出的数据结构，每次操作只需要操作栈顶元素，因此可以将链表的头部元素认为是栈顶元素，压栈和出栈就是头部元素的添加和删除操作

    # (二)：链表实现队列

    • 由于队列是一个先进后出的队列，如果使用之前的链表实现，那么操作尾巴元素比较麻烦，因此，在原有的链表基础上，添加一个tail变量指向最后一个节点，head指向第一个节点，并且去掉了dummyHead虚拟节点来实现队列

    • 实现代码如下：基于链表的思想实现的队列
    // Node 节点
type Node struct {
	e    int   // 数据
	next *Node // 指向的下一个节点
}

// NodeToString 打印节点元素值
func (n *Node) NodeToString() string {
	return strconv.Itoa(n.e)
}

// LinkedListQueue 链表
type LinkedListQueue struct {
	head *Node // 链表指向的第一个节点
	tail *Node // 链表的最后一个节点
	size int   // 链表的长度
}

// NewLinkedListQueue 构造函数
func NewLinkedListQueue() *LinkedListQueue {
	return &LinkedListQueue{
		size: 0,
	}
}

// getSize 获取链表的长度
func (l *LinkedListQueue) getSize() int {
	return l.size
}

// isEmpty 判断链表是否为空
func (l *LinkedListQueue) isEmpty() bool {
	return l.size == 0
}

// enqueue 入队操作, 从尾部进入
func (l *LinkedListQueue) enqueue (e int)  {

	if l.tail == nil {  // 空链表 只需要将新节点赋值给tail即可
		l.tail = &Node{e: e}
		l.head = l.tail

	}else {  // 非空链表 直接将新节点赋值给tail.next, 再让新节点成为当前的tail
		// 移动尾部指针
		l.tail.next = &Node{e: e}
		l.tail = l.tail.next
	}
	l.size++

}

// dequeue 出队操作，从头部出去
func (l *LinkedListQueue) dequeue () int {

	if l.isEmpty(){
		log.Fatalf("此时队列为空")
	}

	// 将节点删除
	delNode := l.head
	l.head = l.head.next
	delNode.next = nil

	// 如果删除节点后，head为空，那么tail也需要设置为nil
	if l.head == nil {
		l.tail = nil
	}

	l.size--

	return delNode.e

}

// getFront 返回第一个元素
func (l *LinkedListQueue) getFront() int {
	if l.isEmpty(){
		log.Fatalf("此时队列为空")
	}
	return l.head.e
}

// toString 方法
func (l *LinkedListQueue) toString() string {
	str := ""

	// 第一个节点
	prev := l.head

	for prev != nil {
		str += prev.NodeToString() + "->"
		prev = prev.next
	}
	// 最后加个nil
	str += "nil"

	return str
}

	linkedListQueue.enqueue(1)
	linkedListQueue.enqueue(2)
	linkedListQueue.enqueue(3)
	linkedListQueue.enqueue(4)

	fmt.Println(linkedListQueue.getFront())
	fmt.Println(linkedListQueue.dequeue())
	fmt.Println(linkedListQueue.toString())

    # 四：链表的其他实现

    # (一)：双向链表

    • 双向链表：Node有两个指针属性，一个是prev，一个是next。具体实现参考 (opens new window)，在Golang中container/list就是双向链表

    • 代码实现
    type DoubleNode struct {
	Data interface{}
	Prev *DoubleNode
	Next *DoubleNode
}

// DoubleList 双链表
type DoubleList struct{
	mutex *sync.RWMutex
	Size uint
	Head *DoubleNode
	Tail *DoubleNode
}

// Init 双链表初始化
func (list *DoubleList)Init()  {
	list.mutex = new(sync.RWMutex)
	list.Size = 0
	list.Head = nil
	list.Tail = nil
}

func (list *DoubleList)Get(index uint) *DoubleNode {
	if list.Size == 0 || index > list.Size - 1 {
		return nil
	}
	if index == 0{
		return list.Head
	}
	node := list.Head
	var i uint
	for i = 1; i <= index; i ++{
		node = node.Next
	}
	return node
}

// Append 添加节点
func (list *DoubleList)Append(node *DoubleNode) bool {
	if node == nil{
		return false
	}

	list.mutex.Lock()
	defer list.mutex.Unlock()

	// 添加头结点
	if list.Size == 0 {
		list.Head = node
		list.Tail = node
		node.Next = nil
		node.Prev = nil

	// 尾部添加节点
	} else {
		node.Prev = list.Tail
		node.Next = nil
		list.Tail.Next = node
		list.Tail = node

	}

	list.Size++
	return true
}

// Insert 向双链表指定位置插入节点
func (list *DoubleList)Insert(index uint, node *DoubleNode) bool {
	if index > list.Size || node == nil{
		return false
	}

	if index == list.Size{
		return list.Append(node)
	}

	list.mutex.Lock()
	defer list.mutex.Unlock()

	// 插入头部节点
	if index == 0{
		node.Next = list.Head
		list.Head = node
		list.Head.Prev = nil
		list.Size++
		return true
	}

	// 中间插入节点
	nextNode := list.Get(index)
	node.Prev = nextNode.Prev
	node.Next = nextNode
	nextNode.Prev.Next = node
	nextNode.Prev = node
	list.Size++
	return true
}

// Delete 删除指定位置的节点
func (list *DoubleList) Delete (index uint) bool {
	if index > list.Size - 1 {
		return false
	}

	list.mutex.Lock()
	defer list.mutex.Unlock()

	// 删除头部节点
	if index == 0 {
		if list.Size == 1{
			list.Head = nil
			list.Tail = nil

		} else {
			node := list.Head
			list.Head.Next.Prev = nil
			list.Head = list.Head.Next
			node.Next = nil

		}
		list.Size--
		return true
	}

	// 删除尾部节点
	if index == list.Size - 1 {
		node := list.Tail
		list.Tail.Prev.Next = nil
		list.Tail = list.Tail.Prev
		node.Prev = nil
		list.Size--
		return true
	}

	node := list.Get(index)
	node.Prev.Next = node.Next
	node.Next.Prev = node.Prev
	node.Prev = nil
	node.Next = nil
	list.Size--
	return true
}

    # (一)：如何将一个链表进行反转

    • 链表反转思想：在链表中，使用pre指向nil，cur指向头部节点【1】，next指向第2个节点【2】。首先将节点【1】的next指针指向nil，然后让pre指向cur，cur指向next，next指向下一个节点，以此类推，直到cur指向nil，那么整个链表就完成了反转

    • 代码实现
    func (l *LinkedList) ReverseList (head *Node) *Node {

    pre := new(Node)
    cur := head

    for cur != nil {
        next := cur.next

        // 让头节点指向nil
        cur.next = pre
        // 让pre指向cur
        pre = cur
        // 让cur指向next
        cur = next

    }

    // 跳出循环的时候，pre就是链表的最后一个节点
    return pre

}
